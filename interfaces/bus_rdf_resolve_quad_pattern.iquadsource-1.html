match

match: (subject: RDF.Term, predicate: RDF.Term, object: RDF.Term, graph: RDF.Term) => AsyncIterator<Quad>
  • Defined in packages/bus-rdf-resolve-quad-pattern/lib/ActorRdfResolveQuadPatternSource.ts:72

Returns a (possibly lazy) stream that processes all quads matching the pattern.

The returned stream MUST expose the property 'metadata'. The implementor is reponsible for handling cases where 'metadata' is being called without the stream being in flow-mode.

param

The exact subject to match, variable is wildcard.

param

The exact predicate to match, variable is wildcard.

param

The exact object to match, variable is wildcard.

param

The exact graph to match, variable is wildcard.

returns

The resulting quad stream.
