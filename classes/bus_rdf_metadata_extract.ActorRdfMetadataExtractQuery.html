Class ActorRdfMetadataExtractQuery

An ActorRdfMetadataExtract that extracts metadata based on a GraphQL-LD query.

It exposes the {@link #queryData} method using which a query can be applied over the metadata stream. For efficiency reasons, the query (and JSON-LD context) must be passed via the actor constructor so that these can be pre-compiled.
