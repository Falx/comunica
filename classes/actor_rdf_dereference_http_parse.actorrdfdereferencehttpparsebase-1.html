Class ActorRdfDereferenceHttpParseBase

An actor that listens on the 'rdf-dereference' bus.

It starts by grabbing all available RDF media types from the RDF parse bus. After that, it resolves the URL using the HTTP bus using an accept header compiled from the available media types. Finally, the response is parsed using the RDF parse bus.
