Class BindingsToQuadsIterator

Transforms a bindings stream into a quad stream given a quad template.

This conforms to the SPARQL 1.1 spec on constructing triples: https://www.w3.org/TR/sparql11-query/#rConstructTriples

Static bindTerm

  • bindTerm(bindings: Bindings, term: RDF.Term): RDF.Term
    • Defined in actor-query-operation-construct/lib/BindingsToQuadsIterator.ts:37

    Bind the given term. If the term is a variable and the variable is bound in the bindings object, return the bindings value. If the term is a variable and the variable is not bound in the bindings object, a falsy value is returned.. Otherwise, the term itself is returned.

    Parameters

    • bindings: Bindings

      A bindings object.

    • term: RDF.Term

      An RDF term.

    Returns RDF.Term

    If the given term is not a variable, the term itself is returned. If the given term is a variable, then the bound term is returned, or a falsy value if it did not exist in the bindings.
