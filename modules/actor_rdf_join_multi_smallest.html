Package actor-rdf-join-multi-smallest

Comunica Multi Smallest RDF Join Actor

A Multi Smallest RDF Join Actor. It accepts 3 or more streams, joins the smallest two, and joins the result with the remaining streams.

This module is part of the Comunica framework.

Install

 
$ yarn add @comunica/actor-rdf-join-multi-smallest
